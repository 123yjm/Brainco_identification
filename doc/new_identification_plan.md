# 三阶段解耦辨识计划

> 核心思想：将"一把梭"的全参数线性回归拆成三个阶段，通过**运动学解耦**把摩擦与惯性分离，再对惯性参数施加**物理可行性约束**，最终得到既有预测精度、又有物理意义的参数集。

---

## 1. 问题分析

### 1.1 当前方法的问题

当前辨识流程（`main_solve.cpp`）对全部 84 个参数做一次性岭回归：

```
min || W·beta - tau ||² + λ||beta||²
```

其中 `beta ∈ R^84` 包含 70 个惯性参数 + 7 个 armature + 7 个 damping。

**根本缺陷**：回归矩阵的列之间存在高度共线性——

- 惯性力矩（`Y_inertial * beta_inertial`）与摩擦力矩（`-qd * damping`）在数值上高度相关
- 优化目标只关心**总力矩预测误差**，不关心单个参数是否物理合理
- solver 可以自由地把力矩"分配"给惯性参数或摩擦参数，只要加起来对上就行

**后果**：力矩 RMSE 可以很低，但单独拿出来的摩擦系数（以及部分惯性参数）完全不可信。

### 1.2 解耦思路

利用不同运动状态下动力学方程的结构差异，分阶段剥离不同参数群：

| 运动状态 | ddq | dq | 主导项 | 可辨识的参数群 |
|----------|-----|-----|--------|---------------|
| Fourier 激励轨迹 | 变化大 | 变化大 | 惯性+摩擦+重力 | 全参数（粗糙初值） |
| 匀速运动 | = 0 | 恒定 | 重力+摩擦 | 摩擦系数 fc, fv |
| 减去摩擦后 | 变化大 | 变化大 | 惯性+重力 | 惯性参数（PF） |

## 2. 实验设计

### 2.1 实验流程

在真机上使用**纯 PD 控制**，不依赖力矩前馈，依次执行以下轨迹：

```
实验 A: Fourier 级数激励轨迹（全关节同时运动）
    ↓
实验 B: 每关节单独匀速运动（两个速度档位 × 7 关节 = 14 段）
```

所有实验均记录 `t, q(1×7), dq(1×7), ddq(1×7), tau(1×7)`。

### 2.2 实验 A：Fourier 激励轨迹

目的：充分激励所有动力学通道，获得第一阶段全参数粗解。

**轨迹形式**（沿用现有 Fourier 框架）：

```
q_i(t) = q_{i,0} + Σ_{k=1}^{N} [ (a_{i,k}/(ω_f k))·sin(ω_f k t) - (b_{i,k}/(ω_f k))·cos(ω_f k t) ]
```

**参数建议**：
- 基频 `ω_f = 2π/T`，周期 `T ≈ 5~10s`（根据机器人力矩裕量调整）
- 谐波数 `N = 5~7`
- 初始条件约束：`q_i(0)=q_{i,0}`, `dq_i(0)=0`, `ddq_i(0)=0`
- 轨迹安全检查：关节限位 + 自碰撞检测（复用现有 `ForceController` 的 Generate-Check-Retry 机制）

**数据量**：建议采集 2~3 个完整周期，采样率 ≥ 500 Hz。

### 2.3 实验 B：匀速运动（摩擦辨识专用）

目的：在 ddq = 0 的条件下，消除惯性项，单独暴露摩擦力。

**对每个关节 j（j = 1..7）**：

1. 将其他 6 个关节锁定在固定位置（避免重力耦合变化）
2. 让关节 j 从 `q_start` 匀速运动到 `q_end`（跨度 ~90°）
3. 速度档位：
   - **低速 v1**：约 5~10 °/s（摩擦主导，重力变化缓慢）
   - **高速 v2**：约 30~60 °/s（增大摩擦信号，但需确保 ddq ≈ 0）

**为什么两个速度就够了**：
- 匀速时（ddq = 0），动力学方程为 `tau = G(q) + fc·sign(dq) + fv·dq`
- 同一位置 q 处，两个不同速度给出两个方程 → 直接解出 fc 和 fv

**关键操作细节**：
- 匀速段的起始和结束需要**加减速缓冲区**（梯形速度 profile），只取中间匀速段的数据
- 正向和反向各跑一次（sign(dq) 分别为 +1 和 -1），用于分离 Coulomb 摩擦的符号
- 每个速度档位采集足够样本（> 100 个有效样本点）

**数据划分**：14 段匀速数据合并为一个数据集 `data_const_vel.csv`。

## 3. 三阶段辨识

### 3.1 第一阶段：全参数粗解

**输入**：实验 A 的 Fourier 轨迹数据

**方法**：标准全参数线性回归（复用现有 `main_solve.cpp` 流程）

```
W_fourier · beta_full = tau_fourier
```

其中：
- `W_fourier`：`(K_fourier × 7) × 84`，含惯性(70) + armature(7) + damping(7)
- 求解 `beta_full`（OLS + Tikhonov 正则化）

**输出**：
- `beta_full`：84 维全参数向量
- 从中提取每个 body 的 `(m, mx, my, mz)` → 用于构造重力项 `G(q)`

**注意**：此阶段得到的 damping 参数**不可信**，仅作为参考。但重力相关参数 `(m, mx, my, mz)` 通常较为可靠，因为重力项 `G(q)` 的函数形式（`sin/cos(q)`）与摩擦项（`dq`）和惯性项（`ddq`）在频域上有较好的分离性。

### 3.2 第二阶段：摩擦系数独立辨识

**核心方程**（匀速运动，ddq = 0）：

```
tau - G(q) = τ_friction(dq)
```

**为什么可以这样做**：
- 匀速时 `ddq = 0` → 惯性项 `M(q)·ddq` 和 `C(q,dq)·dq` 中与 ddq 相关的部分消失
- `G(q)` 不含 `dq` 项 → 重力参数不受第一阶段摩擦辨识误差的影响
- 剩下的 `tau - G(q)` 纯粹是摩擦力

#### 3.2.1 计算 G(q)

利用第一阶段得到的每个 body 的 `(m_i, mx_i, my_i, mz_i)` 构造重力回归矩阵：

对 body i，重力贡献为：

```
τ_gravity,i = m_i · Jv_i^T · g
```

其中 `Jv_i` 是 body i 原点的平移雅可比。这里只需要质量 `m_i` 和一阶矩 `(mx_i, my_i, mz_i)`（用于确定质心位置，进而确定 body 原点的等效重力贡献）。

具体地：

```
G(q) = Σ_i Jv_i(q)^T · m_i · g
```

等价地构造仅含 `(m, mx, my, mz)` 的重力回归矩阵 `Y_gravity`（7×28），则：

```
G(q) = Y_gravity(q, ddq=0, dq=0) · beta_gravity
```

**简化做法**：直接用第一阶段的全参数 `beta_full` 和完整回归矩阵计算：

```
G(q_k) = Y_full(q_k, dq=0, ddq=0) · beta_full
```

这样避免了手动拆分参数。

#### 3.2.2 摩擦模型与求解

采用 **Coulomb + Viscous** 模型（每关节 2 个参数）：

```
τ_friction,j(dq_j) = fc_j · sign(dq_j) + fv_j · dq_j
```

**对每个关节 j**，从匀速数据中提取所有有效样本点 `{q_k, dq_k, tau_k}`（k = 1..K_j），构造：

```
tau_obs,k = tau_k - G(q_k)          // 观测摩擦力
Y_fric,k = [sign(dq_k), dq_k]       // 1×2 回归行
```

堆叠后对关节 j 求解：

```
[fc_j, fv_j]^T = (Y_fric^T · Y_fric)^{-1} · Y_fric^T · tau_obs
```

**使用两个速度档位的数据联合求解**：同一关节的正向 v1、正向 v2、反向 v1、反向 v2 数据合并，四个象限覆盖后 Coulomb 摩擦的符号效应自然被 `sign(dq)` 项捕获。

**物理约束**：
- `fc_j ≥ 0`（Coulomb 摩擦非负）
- `fv_j ≥ 0`（黏性摩擦非负）

若 OLS 解出现负值，做 constrained LS（clamp 到 0 或使用非负最小二乘）。

**输出**：
- `fc = [fc_1, ..., fc_7]`
- `fv = [fv_1, ..., fv_7]`

### 3.3 第三阶段：物理可行惯性参数辨识

#### 3.3.1 从力矩中扣除摩擦

利用第二阶段得到的摩擦系数，对 Fourier 轨迹数据的每一帧扣除摩擦：

```
tau'_k = tau_k - τ_friction(dq_k)
       = tau_k - [fc ◦ sign(dq_k) + fv ◦ dq_k]
```

其中 `◦` 表示逐元素乘法。

#### 3.3.2 裁剪不可辨识的惯性参数

对每个 body，原始 10 个惯性参数：

```
[m, mx, my, mz, Ixx, Ixy, Ixz, Iyy, Iyz, Izz]
```

**裁剪掉交叉惯量项 `Ixy, Ixz, Iyz`**，原因：
- 交叉惯量项在大多数轨迹中辨识性极差（条件数贡献大）
- 它们与对角惯量 `(Ixx, Iyy, Izz)` 高度共线性
- 去除后，惯量张量变为对角形式，自动满足部分物理可行性条件

裁剪后每 body 保留 7 个参数：

```
[m, mx, my, mz, Ixx, Iyy, Izz]
```

总参数维度（以 revoarm_right 为例）：
- 惯性：7 body × 7 = 49
- Armature：7
- 总计：56 个参数

**实现方式**：修改 `computeBodyRegressorBlock()`，或者在构造 W 后直接删除对应列：

```cpp
// 原 84 列 → 删 Ixy(5), Ixz(6), Iyz(8) 每 body → 63 列（惯性）
// 再删 damping(7) → 56 列
// 列索引映射：
//   body i: col 0=m, 1=mx, 2=my, 3=mz, 4=Ixx, (skip 5=Ixy, 6=Ixz), 7=Iyy, (skip 8=Iyz), 9=Izz
// 实际: 保留 0,1,2,3,4,7,9 → 7 columns per body
```

#### 3.3.3 物理可行性约束

对每个 body i，施加以下约束：

**约束 1：质量非负**
```
m_i > 0
```

**约束 2：惯量张量正定**（对角形式下简化为）
```
Ixx_i > 0,  Iyy_i > 0,  Izz_i > 0
```

**约束 3：三角不等式**（任意刚体惯量必须满足）
```
Ixx_i + Iyy_i ≥ Izz_i
Iyy_i + Izz_i ≥ Ixx_i
Izz_i + Ixx_i ≥ Iyy_i
```

**约束 4：质心在合理范围内**（可选，用于抑制 mx/my/mz 漂移）
```
||[mx_i/m_i, my_i/m_i, mz_i/m_i]||₂ ≤ r_max,i
```
其中 `r_max,i` 可以从 CAD 模型或几何包络估计。

**约束 5：Armature 非负**（转子惯量）
```
armature_j ≥ 0
```

#### 3.3.4 约束求解

**方法 A：惩罚法（简单）**

在目标函数中加惩罚项，将约束优化转为无约束：

```
min ||W'·beta' - tau'||² + λ||beta'||² + ρ·Σ penalty(beta')
```

其中 `penalty(·)` 对违反约束的参数施加大的二次惩罚。

优点：实现简单，可复用现有 `solveRegularized()`。
缺点：约束不是硬性的，需要调 `ρ`。

**方法 B：约束二次规划（推荐）**

将问题写成标准 QP 形式：

```
min  ½ beta'^T·(W'^T W' + λI)·beta' - tau'^T W'·beta'
s.t. A_ineq · beta' ≤ b_ineq    // 线性不等式约束
     beta'_lb ≤ beta' ≤ beta'_ub  // 边界约束
```

使用 Eigen 的 `qpOASES` 或 `osqp-eigen` 求解。

**方法 C：重参数化 + 无约束优化**

类似现有 `NonlinearFrictionLM` 的做法，将受约束的参数用无约束变量表示：

```
m = exp(z_m)              → m > 0
Ixx = exp(z_Ixx)          → Ixx > 0
armature = exp(z_arm)     → armature > 0
```

三角不等式通过引入辅助变量或使用 Cholesky 因子参数化 `L·L^T` 来保证正定性。

然后用 LM 求解无约束非线性最小二乘。

#### 3.3.5 输出

| 参数组 | 数量 | 说明 |
|--------|------|------|
| m, mx, my, mz | 28 | 质量和一阶矩（7 body × 4）|
| Ixx, Iyy, Izz | 21 | 对角惯量（7 body × 3）|
| Armature | 7 | 转子惯量 |
| fc | 7 | Coulomb 摩擦（来自第二阶段）|
| fv | 7 | 黏性摩擦（来自第二阶段）|
| **总计** | **70** | |

## 4. 验证方法

### 4.1 力矩预测精度

在独立的验证轨迹（不同于激励轨迹）上评估：

```
RMSE = sqrt( (1/N) · Σ ||tau_pred - tau_meas||² )
```

期望：RMSE 与第一阶段可比或略优（因为物理约束排除了不合理的参数组合）。

### 4.2 参数物理合理性检查

- 所有 `m_i > 0`
- 所有 `Ixx_i, Iyy_i, Izz_i > 0`
- 三角不等式成立
- `fc_j ≥ 0`, `fv_j ≥ 0`
- 与 CAD 标称值（如有）的数量级一致

### 4.3 摩擦辨识交叉验证

将第二阶段得到的 fc/fv 应用于 Fourier 轨迹数据，检查：

```
tau - Y_inertial·beta_inertial - tau_friction(dq)
```

的残差是否接近白噪声（无结构性残差）。

### 4.4 重力项自洽性检查

第二阶段用 `G(q)` 扣除重力 → 第三阶段重新辨识 `(m, mx, my, mz)` → 重新计算 `G'(q)`，检查：

```
||G(q) - G'(q)|| 是否足够小
```

如果差异大，说明参数在两次辨识间发生了显著漂移，需要回到第三阶段再次迭代。

## 6. 预期效果

| 指标 | 原方法（一阶段 OLS） | 新方法（三阶段） |
|------|---------------------|-----------------|
| 力矩 RMSE | 低（可能 0.1~0.5 Nm） | 同等或略优 |
| 摩擦系数可信度 | 低（可能为负或不合理） | 高（从匀速数据直接解耦） |
| 惯性参数物理合理性 | 无保证 | 满足正定性、三角不等式 |
| 参数可迁移性 | 差（过拟合到特定轨迹） | 好（物理约束防止过拟合） |

## 7. 潜在风险与对策

| 风险 | 对策 |
|------|------|
| 匀速运动中仍有微小 ddq 波动 | 对 ddq 做阈值过滤，只保留 `|ddq| < ε` 的样本 |
| G(q) 依赖第一阶段不准的 m/mx/my/mz | 第一阶段的 m/mx/my/mz 通常可靠（重力信号频域独立）；若不放心可在阶段 2 和阶段 3 之间迭代一次 |
| 剔除 Ixy/Ixz/Iyz 导致拟合精度下降 | 在物理可行性允许范围内保留一项（如仅剔除 Ixz/Iyz，保留 Ixy） |
| 约束 QP 求解器引入额外依赖 | 优先用重参数化 + LM，不引入新库 |
| 匀速运动覆盖的 q 范围有限 | 每个关节在多个起始位置重复匀速实验，扩大 q 覆盖率 |
